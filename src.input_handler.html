src.input_handler.utils.conv26(inp: int, len: int) → List¶

Convert a integer in base 26

inpint

input integer, must be <= 10000

lenint

len+1 is the length of the decomposition

List

Decomposition of the input in base 26

Examples:

int2str(15,2) -> [0, 0, 15]

int2str(27,2) -> [0, 1, 1]

int2str(1e4,2) -> [14, 20, 16]
